1

Subir la carpeta seo-engine/

Sube la carpeta seo-engine/ completa al directorio raiz de tu sitio web via FTP o panel de hosting.

Resultado esperado: tu-sitio.com/seo-engine/router.php debe existir.

2

Subir componentes/ del deployment

Sube la carpeta componentes/ generada por el deployment si tu integracion usa componentes.

No reemplaces tu carpeta raiz assets/: en SEO General la configuracion viaja dentro de seo-engine/site-config.json.

3

Reemplazar .htaccess

Este deployment ya incluye un archivo .htaccess completo en la raiz, listo para reemplazar el actual segun la configuracion elegida en Deployment.

Vista previa del contenido generado:

RewriteEngine On # Redirigir de sin www a www (dinamico para cualquier dominio) RewriteCond %{HTTP_HOST} !^www\. [NC] RewriteRule ^(.*)$ https://www.%{HTTP_HOST}/$1 [L,R=301] # Forzar HTTPS (excepto para validaciones SSL) RewriteCond %{HTTPS} off RewriteCond %{REQUEST_URI} !^/\.well-known/acme-challenge/[0-9a-zA-Z_-]+$ RewriteCond %{REQUEST_URI} !^/\.well-known/cpanel-dcv/[0-9a-zA-Z_-]+$ RewriteCond %{REQUEST_URI} !^/\.well-known/pki-validation/[A-F0-9]{32}\.txt(?:\ Comodo\ DCV)?$ RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301] RewriteBase / # No tocar llamadas internas al sistema SEO-Engine RewriteRule ^seo-engine/ - [L,NC] # Ignorar archivos y directorios reales RewriteCond %{REQUEST_FILENAME} -f [OR] RewriteCond %{REQUEST_FILENAME} -d RewriteRule ^ - [L] # Nivel 2: /categoria/seccion/ RewriteCond %{DOCUMENT_ROOT}/seo-engine/pages/n2--$1--$2.json -f RewriteRule ^([a-zA-Z0-9_-]+)/([a-zA-Z0-9_-]+)/?$ seo-engine/router.php?slug1=$1&slug2=$2 [L,QSA,NC] # Nivel 1: /categoria/ RewriteCond %{DOCUMENT_ROOT}/seo-engine/pages/n1--$1.json -f RewriteRule ^([a-zA-Z0-9_-]+)/?$ seo-engine/router.php?slug1=$1 [L,QSA,NC] # Reescribir URL para servicio.php RewriteRule ^([a-zA-Z0-9_-]+)/?$ servicio.php?servicio=$1 [L,QSA]
4

Verificar

Comprueba que las URLs funcionen correctamente:

  • La carpeta seo-engine/ esta en la raiz del sitio
  • El .htaccess fue reemplazado por el archivo incluido en el deployment
  • El .htaccess incluido contempla compatibilidad con servicio.php
  • site-config.json existe dentro de seo-engine/ y no reemplazaste assets/ raiz
  • Las URLs /{categoria}/ cargan las paginas SEO
  • La navegacion del sitio muestra los enlaces SEO
